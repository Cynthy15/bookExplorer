# Book Explorer

Book Explorer is a modern web application that allows users to browse and explore books in a clean and responsive interface.

Live Demo: https://book-explorer-mu.vercel.app/

---

## Features

- Browse books
- Search functionality
- Categories page
- Favorites page
- Fully responsive design
- Modern UI built with Tailwind CSS
- Mobile-friendly navigation menu

---

## Built With

- HTML5
- Tailwind CSS
- JavaScript (ES6 Modules)
- Vercel (Deployment)

## Installation & Setup

If you want to run the project locally:

1. Clone the repository:
git clone https://github.com/your-username/book-explorer.git

2. Open the folder:
cd book-explorer

3. Open `index.html` in your browser  
   OR use Live Server (recommended)
